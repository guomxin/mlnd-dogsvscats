# 猫狗大战
## 描述
使用深度学习方法识别一张图片是猫还是狗：
* 输入：一张彩色图片
* 输出：是猫还是狗
## 数据
数据集可以从 kaggle 上下载。
[Dogs vs. Cats Redux: Kernels Edition](https://www.kaggle.com/c/dogs-vs-cats-redux-kernels-edition/data)

## 实验环境

```
-- python 3.6.8
-- cuda 9.0
-- tensorflow-gpu (1.12.0)
-- Keras (2.2.4)
-- Ubuntu 18.04.1 LTS
-- gpu gtx 1080ti
-- Intel(R) Core(TM) i7-7800X CPU @ 3.50GHz
```
